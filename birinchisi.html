JavaScript bilan tanishuv

JavaScript kursini boshlashdan oldin quyidagi qiziqarli faktlar bilan tanishib chiqing:

  • JavaScript dunyodagi eng keng tarqalgan dasturlash tillaridan biridir.
  • JavaScriptni yuklab olish shart emas. U allaqachon sizning kompyuteringiz brauzeriga o'rnatilgan.
  • JavaScript hamma uchun bepul.
  • JavaScriptni urganish oson.
  • JavaScript yordamida vebsahifalarni interaktiv (muloqotkor) qilib yasash mumkin.
  • JavaScript veb dasturchilar urganishi shart bo'lgan 3 ta dasturlash tilidan biridir. HTML, CSS, JavaScript

JavaScript HTML elementni yashirish mumkin

JavaScript yordamida HTML elementni o'chirish mumkin.

Dasturlashda yuqotish va yashirish bir-biridan farq qiluvchi tushuncha buni bir-biri bilan adashtirib yubormang. Masalan: display: none; elementni yo'qotadi, element o'z joyini boshqa elementga bo'shatib beradi, lekin element butunlay o'chib ketmaydi faqatgina foydalanuvchiga ko'rinmaydi xolos. visibility: hidden; elementni yashiradi, element foydalanuvchiga ko'rinmaydi lekin element o'z joyini boshqa elementga bo'shatib bermaydi va o'z joyini saqlab qoladi.

Quyida JavaScript yordamida HTML elementni yo'qotish (display:none) va yashirishga (visibility: hidden) misol keltiramiz: 


    <p id="ohchirish">
        Tugma bosilganda ushbu element o'chiriladi.
    </p>
    <button 
        onclick="document.getElementById('ohchirish').style.display='none'">
        O'chir
    </button>
    <p id="yashirish">
        Tugma bosilganda ushbu element yashiriladi.
    </p>
    <button 
        onclick="document.getElementById('yashirish').style.visibility='hidden'">
        Yashir
    </button>
