JavaScript chiqishi (output)

JavaScript kodining natijasi quyidagicha 4 ta ko'rinishda namoyon bo'ladi:

  • innerHTML orqali HTML elementining ichida
  • document.write() orqali HTMLning uchida
  • window.alert() orqali ogohlantirish oynasida
  • console.log() orqali brauzerning konsolida

innerHTML orqali JavaScriptni HTML elementining ichida paydo qilish.

innerHTML ni JavaScript kursining 1-chi darsida qisqacha ko'rganmiz. JavaScriptda metodlar foydalaniladi. Shunday metodlardan biri getElementById(). Ushbu metod yordamida HTML tarkibini o'zgartirish yoki yangi ma'lumot kiritish mumkin. Masalan, quyidagi misolda keltirilgan JavaScript kod id hossasi "misolUchun" ga teng bo'lgan HTML elementini topib uning tarkibiga (innerHTML) ma'lumot kiritadi: 


    <p> 
        3 soatda necha minut bor?  
    </p> 
    <p id="misolUchun"></p> 
    <button 
        type="button" 
        onclick='document.getElementById("misolUchun").innerHTML = 3 * 60'
    >
        Javobni ko'rsat! 
    </button>

3 soatda necha minut bor?

innerHTML HTMLga JavaScript yordamida ma'lumot kiritishning eng ko'p tarqalgan usuli hisoblanadi.

document.write() orqali HTMLning ichida ma'lumot yozish.

document.write() ko'p qo'llanilmasada uni urganib qo'yish ziyon qilmaydi. Aslida ushbu usuldan foydalanish maslahat berilmaydi. Quyidagi misolga qarang: 


  <!DOCTYPE html>
  <html>
  <body>
    <h1>Mening vebsahifam</h1>
    <p>Birinchi paragraf.</p>
    <script>
      document.write(5 + 6);
    </script>
  </body>
  </html>
HTML/JS

JavaScript script teglari orasida joylashgan yuqoridagi kod brauzerda quyidagicha ko'rinadi:

Mening vebsahifam

Birinchi paragraf.

Ushbu misolda JavaScript HTML bilan birga yuklanadi. Agar JavaScript HTML yuklanib bo'lgandan keyin ishga tushsa butun HTML kodini o'chirib yuboradi. Masalan: 


  <!DOCTYPE html>
  <html>
  <body>
    <h1>Mening vebsahifam</h1>
    <p>Birinchi paragraf.</p>
    <button 
      type="button" onclick="document.write(5+6)"> Tekshirib ko'ring
    </button>
  </body>
  </html>
HTML/JS

HTML dokument yaratib ushbu kodni qanday ishlashini o'zingiz mustaqil tekshirib ko'ring.

JavaScriptni window.alert() orqali ogohlantirish oynasiga chiqarish

Ogohlantirish oynasi ingliz tilida alert box deyiladi. alert boks deb o'qiladi Quyidagi misolga qarang: 


  ... 
  <button 
  type="button" onclick="window.alert(10-2)">10-2=?</button>
  ...
HTML/JS

Ogohlantiruvchi oynani (alert box) ko'rish uchun quyidagi tugmani bosib ko'ring:

window kalit so'zini tushurib qoldirish mumkin. Buni o'zingiz mustaqil tekshirib ko'ring

console.log()

Kodni xatoliklardan xalos qilish (debugging) maqsadida ma'lumotlarni brauzer konsoliga chiqarish uchun console.log() metodidan foydalanish mumkin. Masalan: 


  <!DOCTYPE html>
  <html>
  <body>
    <h1>Mening vebsahifam</h1>
    <p>Birinchi paragraf.</p>
    <script>
      console.log(100 / 4);
    </script>
  </body>
  </html>
HTML/JS

Brauzer konsolini ochish uchun:

  1. F12 tugmasini bosing
  2. Brauzerning pastki qismida ochilgan oynada yuqori qismida menyuda console tugmasini topib bosing.

Konsolga keyingi darslarda yana qaytamiz.

Esda turting: JavaScriptda matn qo'shtirniq ichida yoziladi. Raqam esa qo'shtirnoqsiz yoziladi. Bu haqda keyingi darslarda to'liqroq urganamiz.

JavaScriptda hech qanday chop qilish (printerga chiqarish) obyekti yoki medodi yuq. JavaScript bilan tashqi chop qiluvchi qurilmalarga (printer) bog'lanib bo'lmaydi. Faqatgina window.print() metodi orqali joriy ekranni (displayni) printerga chiqarish mumkin. Masalan: 

  ...
  <button onclick="window.print()">ushbu sahifani chop qilish</button>
  ...
