JavaScript bilan tanishuv

JavaScript kursini boshlashdan oldin quyidagi qiziqarli faktlar bilan tanishib chiqing:

  • JavaScript dunyodagi eng keng tarqalgan dasturlash tillaridan biridir.
  • JavaScriptni yuklab olish shart emas. U allaqachon sizning kompyuteringiz brauzeriga o'rnatilgan.
  • JavaScript hamma uchun bepul.
  • JavaScriptni urganish oson.
  • JavaScript yordamida vebsahifalarni interaktiv (muloqotkor) qilib yasash mumkin.
  • JavaScript veb dasturchilar urganishi shart bo'lgan 3 ta dasturlash tilidan biridir. HTML, CSS, JavaScript

JavaScript qayerga va qanday yoziladi?

JavaScriptni xuddi CSS kabi HTML kodi ichiga joylash mumkin. Buning uchun uni <script> and </script> teglari orasiga joylash kerak. 

  
    <script>
      document.getElementById("misolUchun").innerHTML = "1-chi JavaScript kodim";
    </script>
  HTML/JS

JavaScript kodini HTMLning <head> qismida yozish mumkin:

  
    <!DOCTYPE html>
    <html>
    <head>
      <script>
        function meningFunksiyam() {
          document.getElementById("misolUchun").innerHTML = "yangi matn";
        }
      </script>
    </head>
    <body>
      <p id="misolUchun"> Tugma bosilganda ushbu matn o'zgarib qoladi.</p>
      <button type="button" onlick="meningFunksiyam()">
         Bosib ko'r 
      </button>
    </body>
    </html>
  HTML/JS

Yuqoridagi kodda "function" kalit so'zi ishlatilgan. Function nimaligini tushunib olish uchun kundalik hayotdan misol keltiramiz: Masalan sizga boshliq topshiriq berdi, topshiriq bir necha qismdan iborat bo'lishi mumkin, masalan, biron joyga borish, kim bilandir uchrashish, nimanidir sotib olish va va hokazorni o'z ochiga olishi mumkin. Lekin bular bari bitta topshiriq hisoblanadi. Function fankshn deb o'qiladi ingliz tilidan o'zbek tiliga "funksiya", "vazifa" deb tarjima qilinadi. JavaScriptda vazifa yaratish uchun "function" kalit so'zi yoziladi va vazifaga nom beriladi. JavaScriptda vazifa chaqirilmaguncha ishlamay turadi, ya'ni vazifa buyruq berilmaguncha bajarilmaydi. Yuqoridagi kodda vazifaga (funksiyaga) "meningFunksiyam" deb nom berilgan.

onclick JavaScriptda voqea hossasi hisoblanadi. onclick foydalanuvchi sichqoncha tugmasini tugmaga (elementni) bosganda ishga tushadi va agar unga funksiya (vazifa) tayinlangan bo'lsa o'sha vazifa ishga tushadi. Yuqoridagi kodda button elementi ichida onclick hodisa hossasi (atributi) yozilgan va u "meningFunksiyam" vazifasiga tayinlangan. Bu degani tugma bosilganda "meningFunksiyam" ishga tushadi.

JavaScript kod HTMLning body qismida joylashtirilganda, uni barcha teglardan so'ng, body qismining eng so'ngida joylash maslahat beriladi. Sababi brauzer yuqoridan pastga qarab o'qiydi, va JavaScript o'qishga vaqt sarflamaydi va birinchi bo'lib kontentni foydalanuvchiga ko'rsatadi va keyin JavaScript kodini o'qiydi.

JavaScript kodini HTMLning ham body ham head qismida joylash ham mumkin.

Huddi CSSni alohida faylda yozish mumkin bo'lgani kabi JavaScriptni ham alohida faylda yozish mumkin. Faylga istalgan nom berish mumkin lekin faylning qo'shimchasi .js bo'lishi kerak. Masalan:

JavaScript kado alohida faylga yozilganda <script> tegi yozilmaydi.

Tashqi .js JavaScript faylini HTMl faylga bog'lash uchun <script> yaratiladi va va src hossasiga tashqi .js fayli manzili bilan nomi yoziladi. Masalan: 

  
    <script src="meningFunksiyam.js"></script>
  HTML

src hossasiga beriladigan manzil 3 xil bo'lishi mumkin:

  • To'liq vebsahifa manzili
  • To'liq fayl manzili
  • Faqat fayl nomi

Bog'lovchi <script> tegi HTMLning head va yoki body qismida joylashishi mumkin.

Alohida faylda yozilgan JavaScript kodining avzalliklari:

  • HTML kodi va JavaScript kodi arashib, chalkashib ketmaydi, alohida turadi.
  • HTML va JavaScript kodlari o'qishga va o'zgartirishga osonlashadi.
  • Vebsahifa ochilish va ishlash tezligini biroz oshiradi.
  • Bitta .js faylini bir-nechta .html faylida qo'llash mumkin. Takrorlanishlarni oldini oladi.

Izoh

Izoh yozilgan kodni izohlash uchun ishlatiladi. HTML va CSS da bo'lgani kabi JavaScriptda izoh foydalanuvchiga ko'rinmaydi. Bir qatorli izoh yozish uchun ikkita oldinga kesim belgisi (ingliz tilida farqard slash deyiladi forvort slesh deb o'qiladi.) ishlatiladi. Masalan: 

  
    //matnni kattalashtiradi
    document.getElementbyId('idHossaNommi').style.fontSize='3rem';
  
    //Ogohlantiruvchi oynaga xabar chiqarish 
    alert("Ehtiyot bo'ling");
  
    console.log(4+5*3); //arifmetik amallar javobi
  JS

Agar kod nima bajarishini esdan chiqmasin desangiz kod yoniga izoh qoldirish mumkin. Agar bitta kod ustida bir necha dasturchi ishlasa, dasturchilar bir biriga kod yonida xabar qoldirishi mumkin. Maslan, "kodni o'zgartirdim, hech kim tegmasin". Izohlardan doimo foydalanish kerak.

Ko'p qatorli izoh yozish uchun oldinga kesim yulduzcha (/*) va yulduzcha oldinga kesim (*/) ishlatiladi. Maslan: 

  
    //matnni kattalashtiradi
    document.getElementbyId('idHossaNommi').style.fontSize='3rem';
  
    /*Ogohlantiruvchi oynaga xabar chiqarish 
    alert("Ehtiyot bo'ling");
  
    arifmetik amallar javobi
    console.log(4+5*3) */;
  JS

Yuqoridagi misolda JavaScript kodi izohga aylantirilgan. Izohga aylantirilgan JavaScript kod ishlamaydi.

Bayonot (Statement)

Kompyuter dasturi kompyuter tomonidan bajarilishi kerak bo'lgan bir qator "topshiriqlardan" iborat. Ushbu topshiriqlar dasturlash tilida bayonot deyiladi. Ingliz tilida statement deyiladi steytment deb o'qiladi JavaScript ham bayonotlardan iborat. HTMLda JavaScript dasturi brauzer tomonidan bajariladi. JavaScript bayonoti quyidagilardan iborat:

  • Ko'satgichlar (value)
  • Amallar (Operatorlar)
  • Ifoda (Expressions)
  • Kalit so'zlar (Keywords)
  • Izohlar (Comments)

JavaScript dasturi odatda ko'p bayonotlardan iborat bo'ladi. Ushbu bayonotlar joylashuviga qarab birin ketin bajariladi.

Har bir bajarsa bo'ladigan bayonotdan so'ng nuqta vergul (;) qo'yiladi. Masalan: 

  
    document.getElementbyId('idHossaNommi').style.fontSize='3rem';
    alert("Ehtiyot bo'ling");
    console.log(4+5*3);
  JS

Nuqta vergul (;) bilan ajratilganda bayonotlarni bir qatorga ketma-ket joylashtirish mumkin. Masalan: 

  
    document.getElementbyId('idHossaNommi').style.fontSize='3rem';
    alert("Ehtiyot bo'ling"); console.log(4+5*3);
  JS

Bayonot oxirida nuqta vergul qo'yish maslahat beriladi lekin majburiy emas.

JavaScript ortiqcha qo'yilgan bo'shliqni e'tiborga olmaydi. Masalan quyidagi ikkita bayonot bir xil kuchga ega: 

  
    alert("Ehtiyot bo'ling");
    alert ( " Ehtiyot bo'ling " );
  JS

JavaScriptda bayonotlarni chimchilangan qavslar ichida bir to'plamga o'rash mumkin. JavaScript bayonotlarini bir to'plamga o'rashdan maqsad ularni bar vaqtda bajarilishini taminlash. Quyida function kalit so'zi yordamida JavaScript bayonotlari bir tudaga o'ralgan: 

  
    function meningFunksiyam {
      alert(99 + 101);
      console.log('Salom');
    }
  JS

function kalit so'zi bilan funksiya yasalganda albatta funksiyaga nom berish shart. Yuqoridagi misolda funksiya "meningFunksiyam" deb nomlangan. Chunki bir nechta funksiya yasalganda ular bir-biridan ajralib turishi kerak. Funksiyalar haqida keyingi darslarda to'liqroq urganamiz.

E'tibor bergan bo'lsangiz yuqoridagii kodda bayonotlar function kalit so'ziga nisbatan chapdan bo'shliq tashlab ketilgan holda yozilgan. Bu ingliz tilida indentation deyiladi, indenteyshn deb o'qiladi o'zbek tiliga chekinish deb tarjima qilinadi.

function kalit so'zidan boshqa JavaScriptda quyidagi kalit so'zlar bor:

Kalit so'z Ma'nosi va vazifasi
var O'zgaruvchi e'lon qiladi
let Blok o'zgaruvchi e'lon qiladi
const blok o'zgarmas e'lon qiladi
if Shartli ravishda bajariladigan bayonotlar to'plami
switch Birin ketin bajariladigan bayonotlar to'plami
loop Takror bajariladigan bayonotlar to'plami
return funksiyani tugatib chiqish
try bayonotlar to'plamiga xatolikni eplashni joriy qiladi

Kalit so'zlar zaxiraga olingan so'zlar hisoblanadi va o'zgaruvchilar uchun nom bo'la olmaydi. Kalit so'zlarni keyingi darslarda chuqurroq urganamiz.

O'zgaruvchi (veriebl) nima o'zi?

O'zgaruvchilar ingliz tilida "variable" deyiladi, veriebl deb o'qiladi Ushbu kursda biz bazan "o'zgaruvchi" bazan "veriebl" deb ataymiz. Ikkalasi bir narsa. Dasturlashda ko'pincha o'zgaruvchan qiymatlarga duch kelamiz. Masalan bank hisob raqami balansi, yoki kimningdur yoshi o'zgaruvchan bo'lishi mumkin. Veriebl o'zgaruvchi ko'rsatgichlarni saqlash uchun bir katak vazafisini bajaradi. Dasturlashda o'zgaruvchi yaratish uchun let kalit so'zidan foydalanish mumkin. Masalan: 

  
    let foydalanuvchiBalansi = 10000;
  JS

O'zgaruvchilarga faqat raqam emas matn ham belgilash mumkin. Masalan: 

  
    let foydalanuvchiBalansi = 10000;
    let foydalanuvchiIsmi = "Azamat";
  JS

O'zgaruvchiga matn belgilaganda qo'shtirnoq ("") yoki yakkatirnoq ('') qo'yish shart.

Agar lozim bo'lsa o'zgaruvchilarni ko'rsatgich bermasdan ham yasab ularga ko'rsatgich keyinchalik berish mumkin. Masalan: 

  
    let x; //"x" nomli o'zgaruvchi yaratildi.
    let y; //"y" nomli o'zgaruvchi yaratildi.
    let a; //"a" nomli o'zgaruvchi yaratildi.
    x = 15; //"x" nomli o'zgaruvchi 15 deb belgilandi.
    a = 25; //"a" nomli o'zgaruvchi 25 deb belgilandi.tortinchisi1
  JS

tenglik belgisi (=) tayinlovchi amal (operator) hisoblanadi.

Agar o'zgaruvchini konsolga chiqarmoqchi bo'lsangiz faqat uni nomini yozish kifoya. Masalan: 

  
    let x;
    let y;
    let a;
    x = 15;
    a = 25;
    console.log(a)
  JS

O'zgaruvchilarni yo'l-yo'lakay o'zgartirib borish mumkin. Masalan: 

  
    <script>
      let foydalanuvchiYoshi = 14;
      /* bir yil o'tdi */
      foydalanuvchiYoshi = 15;
    </script>
    <button onclick = "alert(foydalanuvchiYoshi)">
      Meni bos!
    </button">
  JS

O'zgaruvchilarni istalgancha o'zgartirish mumkin.

JavaScriptnining eski versiyalarida (1995 yildan to 2015 gacha).o'zgaruvchilarni var kalit so'zi bilan yaratishgan: 

  
    var meva = "olma";
  JS

Agar vebsahifangiz eski brauzerlarda ham ishlashi shart bo'lsa unda var kalit so'zidan foydalaning. var kalit so'zidan JavaScriptnining oxirgi versiyalarida ham foydalanish mumkin, lekin maslahat berilmaydi. let kalit so'zi JavaScriptga 2015 yilda kiritilgan va bugungi kunda var kalit so'zini o'rnini egallagan. Chunki var kalit so'zining o'ziga yarasha kamchiliklari bor lekin hozircha bunga chuqurlashmaymiz.

Ohzgarmas

Ba'zida o'zgaruvchiga belgilangan ko'rsatgichni o'zgarmas qilib tayinlashga to'g'ri keladi. Masalan sizning yoshingiz doimo o'zgaradi lekin tug'ilgan yilingiz o'zgarmaydi. Bunday xollarda o'zgaruvchi const kalit so'zi bilan konstant (ohzgarmas) o'zgaruvchi yaratiladi. Masalan: 

  
    let foydalanuvchiYoshi = 15;
    const foydalanuvchiTughilganYili = 1999;
  JS

Konstantlar yaratilganda (elon qilinganda) ko'rsatgich olishi kerak. Agar konstant ko'rsatgichsiz yaratilsa keyinchalik unga ko'rsatgich belgilab bo'lmaydi. Ko'rsatgichli konstantni o'zgartirishga harakat qilsak xatotolik (error) kelib chiqadi. Masalan: 

  
    <script>
      const pasportBerilganYili; /*konstant ko'rsatgichsiz yasaldi 
      const gachaAmalQiladi = 2030; /*konstant ko'rsatgich bilan yasaldi 
      /* Endi birinchi konstantga ko'rsatgich tayinlaymiz va 
      ikkinchi konstantni ko'rsatgichini o'zgartiramiz. */
      pasportBerilganYili = 2020;
      gachaAmalQiladi = 2040;
    </script>
    <button onclick = "alert(pasportBerilganYili)">
      Pasport Berilgan Yili
    </button>
    <button onclick = "alert(gachaAmalQiladi)">
      Amal Qilish Muddati
    </button>
  JS

Ushbu tugmalar ishlamaydi, chunki kodda xatolik bor. Ushbu xatolikni ko'rish uchun F12 tugmasini bosib xato tozalagichni (debager) oching va menyudan "console" bosib konsolni oching.

Konstantlar (o'zgarmaslar) juda kerakli va foydali hisoblanadi.

O'zgaruvchilarni nomlash qoidalari

JavaScriptda nomlashning quidagi qoidalariga amal qiling!:

  • O'zgaruvchilar nomlari harflar, pastki chiziq ( _ ) yoki dollar belgisi ($) bilan boshlanishi kerak.
  • O'zgaruvchilar nomalarida bo'shliq qo'llanib bo'lmaydi.
  • O'zgaruvchi nomlari faqat harflar, raqamlar, pastki chiziq va dollar belgisidan tashkil topishi mumkin.
  • O'zgaruvchi nomlarida bosh harf va kichik harf farq qiladi, masalan, foydalanuvchiIsmi va FoydalanuvchiIsmi bir biridan farq qiladi.
  • Ikkita o'zgaruvchilarga bir xil nom berish mumkin emas.

5-chi dars:
Arifmetik amallar (operatorlar)

JavaScriptda arifmetik amallar qo'shuv (+), ayiruv (-), bo'luv (:) va ko'paytiruv va hokazolarga o'xshash bo'ladi.

Arifmetik amallar.

Arifmetik amallarni arifmetik operatorlar deb ham atash mumkin. JavaScriptda arifmetik amallarning quyidagi turlari mavjud:

  • + qo'shish
  • - ayirish
  • * ko'paytirish
  • ** darajaga ko'paytirish
  • / bo'lish
  • % qoldiq (modulus)
  • ++ oshirish
  • -- kamaytirish

Quyidagi misolga qarang:

  
    <button type="button" onclick="alert(25 + 25)">
      25+25=? 
    </button>
    <button type="button" onclick="alert(25 - 15)">
      25-15=? 
    </button>
    <button type="button" onclick="alert(25 * 4)">
      25*4=? 
    </button>
    <button type="button" onclick="alert(5 ** 3)">
      5**3=? 
    </button>
    <button type="button" onclick="alert(100 / 5)">
      100/5=? 
    </button>
    <button type="button" onclick="alert(10 % 3)">
      10%3=? 
    </button>
  HTML/JS

Yuqoridagi arifmetik amallarning yechimini tugmalarni bosib ogohlantiruvchi oynada ko'rishingiz mumkin.

Bitta bayonnomada bir nechta arifmetik amallarni joylashtirish ham mumkin. Masalan: 

  
    <button type="button" onclick="alert(18 + 2 - 10)">
      Bosib ko'r 
    </button>
  HTML/JS

Bir nechta arifmetik amal birdan qo'yilganda arifmetik qoidaga amal qilinadi. Ya'ni ko'paytiruv va bo'lish avval, qo'shish va ayirish keyin bajariladi. Kerak bo'lganda qavs ham ishlatish mumkin. Masalan: 

  
    <button type="button" onclick="alert(5 + 2 * 3)">
      Bosib ko'r 
    </button>
    <button type="button" onclick="alert((5 + 2 ) * 3)">
      Bosib ko'r 
    </button>
  HTML/JS

Oshirish (++) amalini tushunib olish uchun quyidagi misolga qarang: 

  
    <script>
      let foydalanuvchiToplaganBali = 420;
      foydalanuvchiToplaganBali ++;
    </script>
    <button type="button" onclick="alert(foydalanuvchiToplaganBali)">
      Jami bal
    </button>
  HTML/JS

Agar misolda tushunmagan bo'lsangiz o'zbekcha tushuntiramiz: oshirish (++) amali, mavjud raqamga 1 ni qo'shadi.

Quyidagi misolda kamaytirish (--) amalini qanday ishlashini ko'rishimiz mumkin: 

  
    <script>
      let qolganUrinishlarSoni = 4;
      qolganUrinishlarSoni --;
    </script>
    <button type="button" onclick="alert(qolganUrinishlarSoni)">
      Qolgan urinishlar soni
    </button>
  HTML/JS

6-chi dars:
Malumot turlari

Ushbu darsda JavaScriptda mavjud bo'lgan malumot turlari haqida gaplashamiz. Biz o'zgaruvchilarga matn va raqam tayinlab ular haqida biroz tushunchaga ega bo'ldik. Endi ma'lumot turlari haqida chuqurroq urganamiz.

Mundarija:

  • String va sonlar
  • Boolean

Malumot turlari: String va sonlar

JavaScriptda matn "string" deb ataladi. string deb o'qiladi. String har doim qo'shtirnoq ("")(yoki yakkatirnoq ('') ) ichida yoziladi. Agar raqam qo'shtirnoq ichida yozilsa u ham string hisoblanadi. Masalan: 

  
    let x = "yashashdan maqsad"; //bu string
    let y = "42"; //bu ham string
  JS

String ko'rinishidagi raqamlar arifmetik amallari siz kutgan natijasini bermasligi mumkin. Masalan: 

  
    <script>
      let olmasoni = "5";
      let shaftolisoni = "6";
      let jami = olmasoni + shaftolisoni;
    </script>
    <button onclick="alert(jami)">
      jami
    </button>
  HTML/JS

Yuqoridagi misolda natija 11 emas, balki 56. Chunki "5" va "6" raqam emas, ular string.

Raqamlar butun sonlar yoki o'nli kasr ko'rinishida bo'lishi nmumkin. Masalan: 

  
    let MeningBalim = 7;
    let DostimningBali = 6.5;
  JS

O'nli kasrlar vergul bilan emas, nuqta (.) bilan yoziladi.

Booleanlar

Ba'zidan vebsahifada foydalanuvchi "ha" yoki "yo'q" deb javob berishi kerak bo'ladi. Bunday xollarda va chunga o'xshash xollarda booleanlardan foydalaniladi. Boolean ko'rsatgich ikkita bo'ladi:

true;
o'zbek tiliga "to'g'ri" deb tarjima qilinadi tru deb o'qiladi,
false:
o'zbek tiliga "noto'gri" yoki "yolg'on" deb tarjima qilinadi. fols deb o'qiladi 
  
    let htmlniTamomlagan = true;
    let javaScriptniTamomlagan = false;
  JS

Raqamlarda ifodalansa true 1 raqamiga, false 0 raqamiga teng bo'ladi. Lekin xozircha bunga chuqurlashmaymiz. Keyingi darslarda bunga yana qaytamiz.
