Yuqoridagi kodda "function" kalit so'zi ishlatilgan. Function nimaligini tushunib olish uchun kundalik hayotdan misol keltiramiz: Masalan sizga boshliq topshiriq berdi, topshiriq bir necha qismdan iborat bo'lishi mumkin, masalan, biron joyga borish, kim bilandir uchrashish, nimanidir sotib olish va va hokazorni o'z ochiga olishi mumkin. Lekin bular bari bitta topshiriq hisoblanadi. Function fankshn deb o'qiladi ingliz tilidan o'zbek tiliga "funksiya", "vazifa" deb tarjima qilinadi. JavaScriptda vazifa yaratish uchun "function" kalit so'zi yoziladi va vazifaga nom beriladi. JavaScriptda vazifa chaqirilmaguncha ishlamay turadi, ya'ni vazifa buyruq berilmaguncha bajarilmaydi. Yuqoridagi kodda vazifaga (funksiyaga) "meningFunksiyam" deb nom berilgan.

onclick JavaScriptda voqea hossasi hisoblanadi. onclick foydalanuvchi sichqoncha tugmasini tugmaga (elementni) bosganda ishga tushadi va agar unga funksiya (vazifa) tayinlangan bo'lsa o'sha vazifa ishga tushadi. Yuqoridagi kodda button elementi ichida onclick hodisa hossasi (atributi) yozilgan va u "meningFunksiyam" vazifasiga tayinlangan. Bu degani tugma bosilganda "meningFunksiyam" ishga tushadi.

Bayonot (Statement)

Kompyuter dasturi kompyuter tomonidan bajarilishi kerak bo'lgan bir qator "topshiriqlardan" iborat. Ushbu topshiriqlar dasturlash tilida bayonot deyiladi. Ingliz tilida statement deyiladi steytment deb o'qiladi JavaScript ham bayonotlardan iborat. HTMLda JavaScript dasturi brauzer tomonidan bajariladi. JavaScript bayonoti quyidagilardan iborat:

  • Ko'satgichlar (value)
  • Amallar (Operatorlar)
  • Ifoda (Expressions)
  • Kalit so'zlar (Keywords)
  • Izohlar (Comments)

JavaScript dasturi odatda ko'p bayonotlardan iborat bo'ladi. Ushbu bayonotlar joylashuviga qarab birin ketin bajariladi.

Har bir bajarsa bo'ladigan bayonotdan so'ng nuqta vergul (;) qo'yiladi. Masalan: 


  document.getElementbyId('idHossaNommi').style.fontSize='3rem';
  alert("Ehtiyot bo'ling");
  console.log(4+5*3);
JS

Nuqta vergul (;) bilan ajratilganda bayonotlarni bir qatorga ketma-ket joylashtirish mumkin. Masalan: 


  document.getElementbyId('idHossaNommi').style.fontSize='3rem';
  alert("Ehtiyot bo'ling"); console.log(4+5*3);
JS

Bayonot oxirida nuqta vergul qo'yish maslahat beriladi lekin majburiy emas.

JavaScript ortiqcha qo'yilgan bo'shliqni e'tiborga olmaydi. Masalan quyidagi ikkita bayonot bir xil kuchga ega: 


  alert("Ehtiyot bo'ling");
  alert ( " Ehtiyot bo'ling " );
JS

JavaScriptda bayonotlarni chimchilangan qavslar ichida bir to'plamga o'rash mumkin. JavaScript bayonotlarini bir to'plamga o'rashdan maqsad ularni bar vaqtda bajarilishini taminlash. Quyida function kalit so'zi yordamida JavaScript bayonotlari bir tudaga o'ralgan: 


  function meningFunksiyam {
    alert(99 + 101);
    console.log('Salom');
  }
JS

function kalit so'zi bilan funksiya yasalganda albatta funksiyaga nom berish shart. Yuqoridagi misolda funksiya "meningFunksiyam" deb nomlangan. Chunki bir nechta funksiya yasalganda ular bir-biridan ajralib turishi kerak. Funksiyalar haqida keyingi darslarda to'liqroq urganamiz.

E'tibor bergan bo'lsangiz yuqoridagii kodda bayonotlar function kalit so'ziga nisbatan chapdan bo'shliq tashlab ketilgan holda yozilgan. Bu ingliz tilida indentation deyiladi, indenteyshn deb o'qiladi o'zbek tiliga chekinish deb tarjima qilinadi.

O'zgaruvchi (veriebl) nima o'zi?

O'zgaruvchilar ingliz tilida "variable" deyiladi, veriebl deb o'qiladi Ushbu kursda biz bazan "o'zgaruvchi" bazan "veriebl" deb ataymiz. Ikkalasi bir narsa. Dasturlashda ko'pincha o'zgaruvchan qiymatlarga duch kelamiz. Masalan bank hisob raqami balansi, yoki kimningdur yoshi o'zgaruvchan bo'lishi mumkin. Veriebl o'zgaruvchi ko'rsatgichlarni saqlash uchun bir katak vazafisini bajaradi. Dasturlashda o'zgaruvchi yaratish uchun let kalit so'zidan foydalanish mumkin. Masalan: 


  let foydalanuvchiBalansi = 10000;

for loop

for loop looplarning turlaridan biri hisoblanadi. U quyidagicha yoziladi: 


  for (boshlovchi; shart; oxirgi-ifoda) {
    // kod
  }
JS

boshlovchi (ingliz tilida initializer deyiladi (inishialayze deb o'qiladi)) o'zgaruvhi hisoblanadi uning ko'rsatgichi loop necha marta ishlaganiga qarab o'zgarib boradi.
shart (ingliz tilida "condition" deyiladi (kondishn deb o'qiladi)) loop qachon to'xtashi kerakligini belgilaydi. Ya'ni kod qayta-qayta ishashdan to'xtashi shartda belgilanadi.
oxirgi-ifoda (ingliz tilida "final expression" deyiladi, faynl ekspreshn deb o'qiladi) har safar kod amalga oshgandan so'ng amalga oshadi. Ko'pincha shart ishlatiladigan o'zgaruvchini ohib borish uchun ishlatiladi.
kodni bir marta bajarailishi iteration (itereyshn) deyiladi. Quyida 1 dan 10 gacha bo'lgan raqamlarni konsolga chiqarish bilan for loop ga misol keltiramiz: 


  for (let i = 1; i <= 10; i++) {
    console.log(i);
  }
JS

YUqoridagi kodda: 1. i nomli o'zgaruvchi yaratiladi va uning dastlabki ko'rsatgichi 1 qilib belgilanadi. 2. i ning 10 dan kichkina yoki 10 ga tengligi tekshiriladi. 3. console.log i ni konsolga chiqaradi. 4. i++ ishga tushadi ya'ni i = i + 1. 5. i 10 dan kichkina yoki teng ekanligi sharti tekshiriladi va hokazo.

  for (let i = 0; i <= 20; i += 2) {

  }

While loop

while loop quyidagicha yoziladi:


  while(shart) {
    //bajarilishi kerak bo'lgan kod
  }
JS

while loop shart bajrarilsa (shart "true" ga teng bo'lsa) ishlaydi. Quyidagi misolda 1 dan 10 gacha bo'lga sonlarni konsolga chiqaradigan kod yozamiz: 


  let raqam = 1;
  while (raqam <= 10) {
    console.log(raqam);
    raqam ++;
  }
JS

Ushbu kodning natijasini konsolni ochib tekshirib ko'ring. Bundan tashqari 30 dan 20 gacha raqamlarni konsolga chiqaradigan kodni mustaqil yozib, konsolda tekshirib ko'ring.

while loop da kod ishlagan sari shart 'false' ga aylanishi kerak, ask holda 'infinite loop' sodir bo'ladi. Ya'ni kod to'xtamasdan ishlaydi, umuman to'xtamaydi. 'infinite loop' ishlaganda ba'zida kompyuter qizib ketadi. 'infinite' ingliz tilidan o'zbek tiliga 'cheksiz' deb tarjima qilinadi. Masalan usbu kod cheksiz ishlaydi: 


  let raqam = 1;
  while (raqam >= 1) {
    console.log(raqam);
    raqam ++;
  }
JS

'do while loop' takrorlovchilarning yana bir turi. U while loop takrorlovchisiga o'xshaydi ammo 1 ta farqi bor. Shart 'true' bo'lmay qolgandan keyin to'xtaydi. Masalan: 


  let i=5;
  do {  
      console.log(i);
      i++;  
  }
  while (i<5);

for takrorlovchi va while takrorlovchi bir-biriga juda o'xshash. 'for loop' takrorlanish soni aniq bo'lgan xollarda ishlatiladi. 'while loop' esa takrorlanish sharti murakkab bo'lgan xollarda ishlatiladi. Iloji boricha 'for loop' dan foydalanishga harakat qiling chunki u yozishga, o'qishga va tushunishga oson.

Break and Continue (Uzish va Davom ettirish)

Esingizda bo'lsa switch bayonotini urganganimizda 'break' nimaligini urgangan edik. 'break' ingliz tilidan o'zbek tiliga 'uzish' deb tarjima qilinadi. (breik deb o'qiladi) 'break' bayonoti takrorlashni to'xtatish uchun ishlatiladi. Masalan: 


  for(let i=0; i<10;i++) {
    if(i==3) {
      break;
    }
    console.log(i);
  }
JS

Yuqoridagi ushbu kod 0 dan 2 gacha raqamlarni konsolga chiqaradi. i = 3 bo'lganda loop uziladi. Boshqa ishlamaydi. Konsolga hech narsa chiqamaydi.

break bayonotini 'while' tarorlovchi bilan ham ishlatish mumkin. 


  let num = 1;

  while(num<=10){
      if(num == 3){
          break;
      }
      console.log(num);
      num++;
  }
JS

'continue' bayonoti takrorlashda ma'lum shart bajarailgan holatni tashlab ketib davom etishni uchun ishlatiladi. Masalan, mexmonxonada 50 ta xona bor va hammasi bo'sh. Siz 13 raqamini yomon ko'rasiz, va sizga 13 dan boshqa mexmonxonadagi barcha xonalar raqami ro'yxati kerak: 


  for (let xona = 1; xona <= 50; xona ++) {
    console.log(xona)
  }
JS

ushbu kod 13 raqamini tashlab ketgan xolda 1 dan 50 gacha bo'lgan raqamlarni konsolga chiqaradi.

Function (fankshn yoki funksiya)

function ingliz tilidan o'zbek tiliga 'vazifa' deb tarjima qilinadi. (fankshn deb o'qiladi) Ma'lum bir vazifani bajarish uchun yozilgan kodlar toplami fankshn bilan o'raladi. Fankshnga nom beriladi. Keyin har safar kerak bo'lganda fankshn nomi ko'rsatiladi va fankshn ishga tushiriladi. Fankshnni nomini yozib ishga tushirilish, dasturlashda fankshnni chaqirish deyiladi.

Mundarija

  function nom () {

fankshn parametrlari

E'tibor bergan bo'lsangiz fankshn yaratganda fankshn nomidan keyin qavslar () yoziladi. Ushbu qavslar aslida fankshn parametrlari uchun mo'ljallangan. Parametrlardan fankshn ichida veriebl kabi foydalanish mumkin. Quyidagi misolda 'qutlov' nomli fankshnga 'ism' deb nomlangan parametr kiritamiz: 


  function qutlov2 (ism) {
    console.log(`Hurmatli, ${ism} vebsahifamga xush kelibsiz!`)
  }
JS

Parametrli fankshnni chaqirganda uni nomi bilan birga parametr yozish kerak. Maslan yuqoridagi qutlov2 fankshnini quyiagicha chaqiriladi: 


  function qutlov2 (ism) {
    console.log(`Hurmatli, ${ism}. Vebsahifamga xush kelibsiz!`)
  }
  qutlov2 ("Azamat");
  qutlov2 ("Bahora");
JS

Yuqorida misolda ko'rib turganingiz kabi funksiyani har xil parametr bilan chaqirib har xil natija olish mumkin. Fankshn bir necht parametr olishi ham mumkin. Buning uchun ularni vergul (,) bilan ajratish kerak. Masalan: 


  function xabar (ism, balans) {

  xabar (Azamat, 3500);
  xabar (Bahora, 4200);

  function xabar2(ism2, balans2) {

      `Hurmatli ${ism2}. Hisobingizda ${balans2} so'm mavjud.`);
